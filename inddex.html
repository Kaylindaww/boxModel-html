What is css box model

This is a box that wraps around every html
element.It consist of margins,boarders,padding
and the actual contents.

The main properties of a css box model.

Contents.The contents of the box where
text and images appear.

Padding.Clears an area around the content
The padding is transparent.

Boarder.A boarder that goes around the
and contents.

Margin.Clears an area outside the boader.
The margin is also transparent.
